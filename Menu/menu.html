Qué es una API?

Una API (Interfaz de Programación de Aplicaciones) es como un conjunto de reglas que permite que diferentes programas informáticos se comuniquen entre sí de manera organizada y eficiente. Es como un puente que facilita la interacción y el intercambio de información entre distintas aplicaciones.
